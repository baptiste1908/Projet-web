The Room

53 Rue Léon Gambetta, 59000 Lille

The Room est une très bonne boîte de nuit pour tous les rats de France. L’ambiance est assez bonne avec des sons commerciaux. L’entrée est totalement gratuite, ce qui est à souligner. Le prix des consommations est assez raisonnable. Cependant, la fréquentation est vraiment moyenne, voire dangereuse pour les filles. Nous la conseillons si vous souhaitez juste danser.
