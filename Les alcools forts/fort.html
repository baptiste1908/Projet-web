Les alcools Forts

Voici une liste d'alcools fort. Cette liste est évidemment incomplête.

Les alcools forts regroupent les boissons ayant un degré d'alcool supérieur à 18°.

Le Rhum :


  • Descriptif : Alcool de canne à sucre
  • Pays d'origine : Antilles
  • Degré d'alcool : 40 à 62%

    • ☆ ☆ ☆ ☆ ☆

La Vodka :


  • Descriptif : Base de pomme de terre, de betterave
  • Pays d'origine : Russie, Pologne
  • Degré d'alcool : 40%
    • ☆ ☆ ☆ ☆ ☆

Le Whisky :


  • Descriptif : Base de céréales (orge, seigle)
  • Pays d'origine : Écosse, Irlande
  • Degré d'alcool : 40% et plus
    • ☆ ☆ ☆ ☆ ☆

Le Pastis :


  • Descriptif : Base de réglisse
  • Pays d'origine : France
  • Degré d'alcool : 40 à 45%
  • ☆ ☆ ☆ ☆ ☆

Le Gin :


  • Descriptif : Base de céréales
  • Pays d'origine : Pays-Bas
  • Degré d'alcool : 37.5% minimum
    • ☆ ☆ ☆ ☆ ☆
